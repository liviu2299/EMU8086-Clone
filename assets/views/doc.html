Registers

Name Information
A General Register
B General Register
C General Register
D General Register
IP Instruction Pointer

Flags

Name Information
Sign + False / - True
Zero When ALU result is 0
Carry When ALU result exceeds 8bit limit

Instructions

MOV - Copy operand2 to operand1.

  • MOV reg, memory
  • MOV reg, memory
  • MOV reg, regaddress
  • MOV reg, immediate
  • MOV address, reg
  • MOV address, immediate
  • MOV regaddress, reg
  • MOV regaddress, immediate

ADD - Add operands with Carry.

  • ADD reg, immediate
  • ADD reg, reg
  • ADD reg, regaddress
  • ADD reg, address

SUB - Subtract operand2 from operand1.

  • SUB reg, immediate
  • SUB reg, reg
  • SUB reg, regaddress
  • SUB reg, address

INC - Increments operand1.

  • INC reg

DEC - Decrements operand1.

  • DEC reg

CMP - Compare: operand1 - operand2. Results are stored only in flags.

  • CMP reg, immediate
  • CMP reg, reg
  • CMP reg, regaddress
  • CMP reg, address

DIV - Unsigned divide: A / operand1.

  • DIV reg
  • DIV immediate
  • DIV regaddress
  • DIV address

MUL - Unsigned multiply: A * operand1.

  • MUL reg
  • MUL immediate
  • MUL regaddress
  • MUL address

AND - Logical AND between all bits of two operands.

  • AND reg, immediate
  • AND reg, reg
  • AND reg, regaddress
  • AND reg, address

OR - Logical OR between all bits of two operands.

  • OR reg, immediate
  • OR reg, reg
  • OR reg, regaddress
  • OR reg, address

SHR - Shift operand1 to the right operand2 times.

  • SHR reg, immediate

SHL - Shift operand1 to the left operand2 times.

  • SHL reg, immediate

NOT - Invert each bit of the operand.

  • NOT reg

PUSH - Store 8 bit value in the stack.

  • PUSH reg
  • PUSH immediate
  • PUSH regaddress
  • PUSH address

POP - Get 8 bit value from the stack.

  • POP reg

JMP - Unconditional Jump.

  • JMP regaddress
  • JMP address

JG - Jump if operand1 is greater then operand1. (Sign&Zero)

  • JG address

JGE - Jump if operand1 is greater or equal with operand1. (Sign)

  • JGE address

JL - Jump if operand1 is lower then operand1. (Sign&Zero)

  • JL address

JLE - Jump if operand1 is lower or equal with operand1. (Sign)

  • JLE address

JCXZ - Jump if C = 0.

  • JCXZ address

JE - Jump if operand1 is equal to operand1.

  • JE address

JNE - Jump if operand1 is not equal to operand1.

  • JNE address

HLT - Halt the System.

  • no operands

LOOP - Decrease C, jump to label if C != zero.

  • LOOP address
